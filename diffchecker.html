User Guide & Shortcuts

➕ New Tab

  • Opens a blank instance of the editor in a new browser tab.
  • Loading a file while you already have text in the current tab will automatically open the new file in a new tab without overwriting your current work.

💾 Saving Files

  • Save (Ctrl+S): Saves changes directly to the currently loaded file. Note: This requires modern browser support (File System Access API) and a secure context (HTTPS).
  • Save As (Ctrl+Shift+S): Prompts you to choose a new location or filename to download the text.
  • Auto-save: When checked, automatically saves the file to disk every 5 seconds if there are unsaved changes. (Only works if a file is explicitly loaded/saved).

📂 Loading & Importing

  • Click Load to browse your computer or view your recent edit history.
  • You can also Drag & Drop any .txt file directly into the Load menu or anywhere on the screen to load it in a new tab.

🔄 Navigation & Editing

  • Splitter (Resizer): You can drag the vertical divider between the two editors to adjust their widths.
  • Undo/Redo: Reverts or reapplies your text edits inside the active panel.
  • Prev/Next Change: Automatically scrolls to the next or previous highlighted text difference.
  • Empty Line Highlight: Lines that are entirely empty or contain only spaces/tabs are highlighted in a hatched yellow pattern to help you manage formatting. You can toggle this feature On/Off using the toolbar button.

🩹 Auto-Recovery & History

  • The editor automatically caches your work every 5 seconds to prevent accidental data loss.
  • If you close the tab by mistake, you will be prompted to restore your unsaved work the next time you open the editor.
  • Multiple tabs editing different files are safely managed without overwriting each other's cache.
